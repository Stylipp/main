Create feedback feature module following project convention:

**schemas/schemas.py:**
- FeedbackAction(str, Enum): LIKE = "like", DISLIKE = "dislike", SAVE = "save"
- FeedbackRequest(BaseModel): product_id (str — external_id from Qdrant, NOT UUID), action (FeedbackAction)
- FeedbackResponse(BaseModel): id (str), action (str), created_at (datetime)

**service/service.py:**
- FeedbackService class with async method record_feedback(user_id: UUID, product_id_external: str, action: str, session: AsyncSession) -> UserInteraction
- Lookup Product by external_id to get the actual UUID for FK. Raise ValueError if product not found.
- Create UserInteraction record and commit. Do NOT enforce uniqueness — users can interact with the same product multiple times (e.g., dislike then undo → like).

**router/router.py:**
- APIRouter with prefix="/feedback", tags=["Feedback"]
- POST "/" endpoint accepting FeedbackRequest body, using Depends(get_current_user) for auth and Depends(get_db) for session
- Returns FeedbackResponse with 201 status code
- Catches ValueError from service → 404 (product not found)

**main.py:**
- Import feedback_router and register: app.include_router(feedback_router, prefix="/api")

Create __init__.py files in all new directories (empty files).
