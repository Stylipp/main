  <action>
    First, add an `update` method to the existing ProductRepository:

    ```python
    async def update(self, external_id: str, store_id: str, data: dict) -> Product | None:
        """Update an existing product by external_id and store_id.

        Args:
            external_id: The product's external identifier.
            store_id: The store the product belongs to.
            data: Dict of fields to update (title, price, image_url, etc.)

        Returns:
            Updated Product or None if not found.
        """
        product = await self.get_by_external_id(external_id, store_id)
        if product is None:
            return None
        for key, value in data.items():
            if hasattr(product, key):
                setattr(product, key, value)
        await self.session.flush()
        return product
    ```

    Then create DbSyncService in scraper/service/db_sync_service.py:

    Class **DbSyncService**:

    - `__init__(self, session: AsyncSession)` — stores the database session

    - `async def sync_new_products(self, products: list[ScrapedProduct], woo_store_id: str) -> list[Product]`:
      For each ScrapedProduct in the "new" list:
      1. Convert ScrapedProduct → ProductCreate:
         - external_id = product.external_id
         - store_id = woo_store_id
         - title = product.title
         - description = product.description
         - price = product.price (use Decimal("0") if None)
         - currency = product.currency
         - image_url = product.image_urls[0] if product.image_urls else "" (skip if no images)
         - product_url = product.url
      2. Skip products with no image_urls (can't generate embeddings without images)
      3. Create via ProductRepository.create()
      4. Collect and return created Product objects
      5. Log: "{n} new products synced to PostgreSQL"

    - `async def sync_changed_products(self, products: list[ScrapedProduct], woo_store_id: str) -> list[Product]`:
      For each ScrapedProduct in the "changed" list:
      1. Build update dict from ScrapedProduct fields (title, description, price, image_url, product_url)
      2. Call ProductRepository.update(external_id, woo_store_id, update_dict)
      3. Collect and return updated Product objects
      4. Log: "{n} changed products updated in PostgreSQL"

    - `async def remove_products(self, external_ids: list[str], store_id: str) -> int`:
      For now, log removed products but do NOT delete from PostgreSQL (products may still be referenced by user interactions). Just log a warning:
      "WARNING: {n} products removed from store {store_id} sitemap but retained in database"
      Return count. Actual deletion/archival is a future concern.

    Import types: ScrapedProduct from scraper schemas, ProductCreate from products schemas, Product from models, ProductRepository, AsyncSession.
