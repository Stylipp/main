  <name>Task 1: Create async sitemap fetcher and URL extractor</name>
  <files>apps/backend/src/features/scraper/service/sitemap_service.py</files>
  <action>
    Create SitemapService that fetches and parses XML sitemaps to extract product URLs.

    Class **SitemapService**:

    - `__init__(self, timeout: float = 30.0)` — stores timeout config

    - `async def fetch_product_urls(self, store: StoreConfig) -> list[str]`:
      1. Fetch sitemap XML from store.sitemap_url using httpx.AsyncClient
      2. Set User-Agent header: "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36" (same as The Sprapper)
      3. Parse XML with xml.etree.ElementTree (stdlib — NOT lxml, to avoid C dependency issues)
      4. Handle sitemap namespace: `{http://www.sitemaps.org/schemas/sitemap/0.9}` prefix on `<url>` and `<loc>` tags
      5. Extract all `<loc>` text values
      6. Filter to only URLs containing store.product_url_pattern (default "/product/")
      7. Return sorted list of unique product URLs
      8. Log: total URLs found vs product URLs filtered

    - `async def fetch_sitemap_index(self, url: str) -> list[str]`:
      Handle sitemap index files (contain `<sitemap><loc>` entries pointing to sub-sitemaps).
      If the root sitemap contains `<sitemapindex>`, recursively fetch each sub-sitemap and aggregate URLs.
      Most WooCommerce stores use direct product sitemaps, so this is a fallback.

    Error handling:
    - httpx.HTTPStatusError → log warning, return empty list (don't crash pipeline for one store)
    - xml.etree.ElementTree.ParseError → log warning, return empty list
    - Timeout → log warning, return empty list

    Import StoreConfig from scraper.config.stores.

<task type="auto">
  <name>Task 2: Create async product page scraper with JSON-LD + HTML fallback extraction</name>
  <files>apps/backend/src/features/scraper/service/product_scraper.py</files>
  <action>
    Port The Sprapper's product extraction logic to an async service with configurable CSS selectors.

    Class **ProductScraper**:

    - `__init__(self, timeout: float = 30.0)` — stores timeout config

    - `async def scrape_product(self, url: str, store: StoreConfig) -> ScrapedProduct | None`:
      1. Fetch page HTML using httpx.AsyncClient with User-Agent header and store.timeout_seconds
      2. Parse with BeautifulSoup(html, "html.parser")
      3. Try JSON-LD extraction first (_extract_from_jsonld), fall back to HTML selectors (_extract_from_html)
      4. Build and return ScrapedProduct with store.id as store_id
      5. Generate external_id from URL: hashlib.md5(url.encode()).hexdigest()[:16]
      6. Set scraped_at to datetime.utcnow()
      7. On any error → log warning, return None (don't crash pipeline for one product)

    - `def _extract_from_jsonld(self, soup: BeautifulSoup, url: str) -> dict | None`:
      Port from The Sprapper's `extract_product_data_jsonld()`:
      1. Find all `<script type="application/ld+json">` tags
      2. Parse each as JSON
      3. Handle Yoast SEO @graph arrays: iterate items, find type "Product" or containing "Product" in @type
      4. Also handle direct Product JSON-LD (no @graph wrapper)
      5. Extract: name, description, image (handle string or list or dict with @id), offers/price
      6. For offers: handle single offer object or list of offers. Get price from first valid offer.
      7. Return dict with keys: title, description, price, sale_price, image_urls, categories
      8. Return None if no Product found in any JSON-LD block

    - `def _extract_from_html(self, soup: BeautifulSoup, store: StoreConfig) -> dict`:
      Port from The Sprapper's `extract_product_data_html()`:
      1. Title: soup.select_one(store.selectors.title) → text.strip()
      2. Price extraction (from The Sprapper's logic):
         - Try regular price: soup.select_one(selectors.price_regular) for crossed-out price
         - Try sale price: soup.select_one(selectors.price_sale) for discounted price
         - Fallback single price: soup.select_one(selectors.price_single)
         - Clean price text: remove currency symbols (₪, $, €), commas, whitespace. Parse to Decimal.
      3. Description: soup.select_one(selectors.description) → text.strip()[:1000]
      4. Categories: [a.text.strip() for a in soup.select(selectors.categories)] — skip "Home"/"דף הבית" entries
      5. Images: Extract from selectors.images. For WooCommerce, prefer data-large_image attribute, fall back to src. Deduplicate.
      6. Return dict with same keys as JSON-LD extraction

    - `async def scrape_batch(self, urls: list[str], store: StoreConfig) -> list[ScrapedProduct]`:
      Scrape multiple URLs sequentially with rate limiting:
      1. For each URL, call scrape_product()
      2. Sleep asyncio.sleep(store.rate_limit_seconds) between requests
      3. Collect non-None results into list
      4. Log progress every 50 products: "{scraped}/{total} products scraped for {store.name}"
      5. Return list of ScrapedProduct

    Import ScrapedProduct from scraper.schemas.schemas, StoreConfig from scraper.config.stores.

    Key differences from The Sprapper to handle:
    - Use httpx (async) not requests (sync)
    - Use store.selectors (configurable) not hardcoded CSS selectors
    - Skip variation handling (backend Product model is flat — variations become separate products only if they have unique URLs/sitemaps)
    - Don't generate CSV — return ScrapedProduct objects
